# AdServices CLI

This directory contains a tool for interacting with the `adservices` module on
Android.

## Features

*   Enable and disable adservices service, flags, etc...
*   Get the status of adservices process

## Getting started

1.  Install dependencies with `pip3 -r requirements.txt`
    *   `python3` and `pip3` are required. See https://www.python.org/downloads/
        for installation details.
2.  Run `./main.py` to see help commands
3.  Optionally add `alias adservices="python3 path/to/main.py"` to your .bashrc
